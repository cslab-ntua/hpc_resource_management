The project SEANERGYS (Software for Efficient and Energy-Aware Supercomputers) aims at developing a next-generation software solution to improve the energy-efficient operations of EuroHPC supercomputers. 

More information can be found [here](https://www.eurohpc-ju.europa.eu/research-innovation/our-projects/seanergys_en)
